## 🎯 What Claude Config Should Look Like

**File:** `%APPDATA%\Claude\claude_desktop_config.json`

```json
{
  "mcpServers": {
    "jmeter": {
      "command": "py",
      "args": [
        "D:\\Jmeter_MCP_NEW\\server.py"
      ],
      "env": {
        "JMETER_HOME": "D:\\Program Files\\Jmeter\\apache-jmeter-5.6.3",
        "PYTHONPATH": "D:\\Jmeter_MCP_NEW"
      }
    }
  }
}
```

**Key Points:**
- Double backslashes: `\\` not `\`
- All paths must exist on your computer
- Save as `.json` file (not `.txt`)

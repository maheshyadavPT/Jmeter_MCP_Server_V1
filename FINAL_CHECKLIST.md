### Claude Says: "Server connection error"

**Solution:**
1. Verify D:\Jmeter_MCP_NEW\server.py exists
2. Check Python works: `py --version` in PowerShell
3. Check PYTHONPATH: Should be D:\Jmeter_MCP_NEW

**Solution:**
1. Check Claude's exact error message
2. Verify all Python files exist

4. Check Claude Desktop logs: Help → Logs

**Solution:**
1. Verify all .py files in D:\Jmeter_MCP_NEW\ exist
2. Check PYTHONPATH is set correctly in config
3. Run test: `py -c "from generate_plan import generate_test_plan"`
